== Installation ==

1. Log into your WordPress admin
2. Click __Plugins__
3. Click __Add New__
4. Search for __Night Traffic Light__
5. Click __Install Now__ under "Night Traffic Light"
6. Activate the plugin

= Manual installation =

1. Download and unzip the plugin.
2. Upload the `instagram-picture` directory to `/wp-content/plugins/`.
3. Activate the plugin through the 'Plugins' menu in WordPress.
